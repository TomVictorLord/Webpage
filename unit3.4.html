3.4 Describe how you would go about testing digital solutions

I could make a project of my own and set my own deadline for it which would challenge and force me to try something new or seek help to meet it, like starting a group on GitHub.
That way numerous people will be working on the same project simultaneously which would be a lot more efficient than single.
If a team member needs help they could ask on Slack.com, then other team mates could reply and help each other from anywhere which would save a lot of time and make finishing before the deadline more likely.
